HTML

  • HTML stands for Hyper Text Markup Language. It is the standard markup language for creating web pages and web applications. HTML describes the structure of a web page using markup.

  • HTML elements are represented by tags. Tags are the building blocks of HTML and are used to create elements such as headings, paragraphs, links, images, and more.

  • HTML is not a programming language; it is a markup language that defines the structure and presentation of content on the web.

HTML Document Structure

  • Every HTML document starts with a doctype declaration that specifies the version of HTML being used.

  • The <html> tag is the root element of an HTML page.

  • The <head> section contains meta-information about the document, such as the title and links to stylesheets or scripts.

  • The <body> section contains the content of the web page, including text, images, and other media.

Basic HTML Tags

  • <h1> to <h6>: Heading tags used to define headings of different levels.

  • <p>: Paragraph tag used to define blocks of text.

  • <a>: Anchor tag used to create hyperlinks.

  • <img>: Image tag used to embed images in a web page.

  • <div> and <span>: Used for grouping and styling content.

Attributes

  • HTML elements can have attributes that provide additional information about the element.

  • Attributes provide extra information about HTML elementshref="https://example.com" for links.

  • Commonly Used Attributes class, id, and style.

    1. href - Specifies the URL for a link (<a>).
    2. src - Specifies the URL for an image (<img>).
    3. alt - Provides alternative text for an image (<img>).
    4. class - Specifies a class for an element, allowing for CSS styling.
    5. id - Specifies a unique identifier for an element.
    6. style - Allows inline CSS styling for an element.
    7. title - Provides additional information about an element, often displayed as a tooltip.
    8. target - Specifies where to open a linked document (e.g., _blank for a new tab).

HTML Comments

Comments in HTML are used to add notes or explanations within the code. They are not displayed in the browser.

To create a comment, use the following syntax:

    <!-- This is a comment -->

HTML Forms

HTML forms are used to collect user input. They consist of various form elements such as text fields, checkboxes, radio buttons, and submit buttons.

Common form elements include:

  • <input>: Used for various types of input fields (text, password, checkbox, radio, etc.).
  • <textarea>: Used for multi-line text input.
  • <select>: Used to create drop-down lists.
  • <button>: Used to create clickable buttons.
Back to top

HTML Entities

HTML entities are special characters that can be used in HTML to represent characters that have a special meaning in HTML or are not easily typed on a keyboard.

Common HTML entities include:

  • & - Represents the ampersand (&)
  • < - Represents the less-than sign (<)
  • > - Represents the greater-than sign (>)
  • " - Represents double quotes ("")
  • ' - Represents single quotes ('')

HTML Best Practices

To create well-structured and maintainable HTML code, consider the following best practices:

  • Use semantic HTML tags to improve accessibility and SEO.
  • Keep your code organized and properly indented for readability.
  • Use comments to explain complex sections of code.
  • Validate your HTML code using online validators to ensure it meets standards.
  • Test your web pages in different browsers to ensure compatibility.
  • Use external stylesheets and scripts to keep your HTML clean and maintainable.

lists

HTML provides two types of lists: ordered lists and unordered lists.

Ordered lists are used for items that have a specific order, while unordered lists are used for items that do not have a specific order.

Common list elements include:

  • <ul>: Represents an unordered list.
  • <ol>: Represents an ordered list.
  • <li>: Represents a list item within a list.
  • <dl>: Represents a description list.
  • <dt>: Represents a term in a description list.
  • <dd>: Represents a description of a term in a description list.
Back to top

Tables

HTML tables are used to display tabular data. They consist of rows and columns, with each cell containing data.

Common table elements include:

  • <table>: The main container for the table.
  • <tr>: Represents a table row.
  • <th>: Represents a table header cell.
  • <td>: Represents a table data cell.
  • <caption>: Provides a title or caption for the table.
Back to top

HTML Media

HTML supports embedding multimedia content such as images, audio, and video.

Common media elements include:

  • <img>: Used to embed images in a web page.
  • <audio>: Used to embed audio files.
  • <video>: Used to embed video files.
  • <iframe>: Used to embed other web pages or content within a page.
Back to top

Iframes

An iframe (inline frame) is used to embed another HTML document within the current document. It allows you to display content from other sources, such as videos or external web pages.

To create an iframe, use the following syntax:

    <iframe src="URL" width="width" height="height"></iframe>
